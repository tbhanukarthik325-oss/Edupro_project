# 📚 EduPro Analytics Dashboard

## 📖 Project Overview

The EduPro Analytics Dashboard is an interactive data analytics application developed using Python, Streamlit, MySQL, Pandas, and Plotly. It analyzes learner demographics, course enrollments, revenue generation, teacher expertise, and payment methods to provide meaningful business insights for an online learning platform.

The dashboard enables users to interactively explore data using filters and visualize trends through charts and KPIs.

---

## 🎯 Project Objectives

- Analyze learner demographics.
- Understand course enrollment behavior.
- Identify the most popular courses.
- Analyze revenue generated by courses.
- Explore teacher expertise.
- Study payment method preferences.
- Build an interactive analytics dashboard.

---

## 🛠️ Technologies Used

- Python

## 📊 Dashboard Features

- Dashboard Summary KPIs
- Executive Summary
- Sidebar Filters
- Age Distribution
- Gender Distribution
- Course Category Analysis
- Course Level Distribution
- Monthly Enrollment Trend
- Revenue by Course Category
- Revenue by Course Level
- Top 10 Most Popular Courses
- Top 10 Highest Revenue Courses
- Teacher Expertise Analysis
- Age Group vs Course Category
- Gender vs Course Level
- Payment Method Distribution
- Download Filtered Data (CSV)
- Interactive Data Explorer

## 🚀 Installation

1. Clone the repository.

2. Install the required packages:

```bash
pip install -r requirements.txt
```

3. Start the Streamlit application:

```bash
streamlit run app.py
```
